# SerwerWWW

## Opis
SerwerWWW to moja próba nauki dokera. Czasami przydaje sie shostować po lanie parę plików, czemu używać do tego http. Postanowiłem to jakoś zabezpieczyć a przy okazji pobawić sie dokerem.

## Główne Cechy
- 🚀 Szybkość instalacji
- 🔒 W miarę bezpieczny - używa certyfikatów
- ⚡ Wydajny i mało zasobożerny

## Technologie
- Javascript
- HTML
- Docker
- Nginx

## Instalacja
Wystarczy uruchomić skrypt create.sh - potrzebne będą uprawnienia roota do zarządzania dockerowym socketem.
